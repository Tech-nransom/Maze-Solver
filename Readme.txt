*)MAZE SOLVER

Steps:
	You Can Create Your Own Maze In the Text File But make Sure:
	
	1)It Has A Starting Point('A') and Ending Point('B')
	2)It Should Have Exactly 1 starting and Ending Point
	3)You can Use anything as walls But ('#') is preferable

Steps To Run The Program:
	1)Go to Command prompt
	2)python maze.py {mode of operation} {maze filename}

Mode of operation:
1)fast
2)efficient

Maze filename:
The File Which contains The Maze


Example:
	1)python maze.py fast maze.txt
	2)python maze.py efficient maze.txt  

Note:
	Fast means DFS
	Efficient means BFS
